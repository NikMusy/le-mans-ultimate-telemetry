STRATEGIST GUIDE · PIT WALL CHEAT SHEET ×

QUICK START

  1. The driver sends you a link like https://lmu-pitwall.pages.dev/?ws=wss://...trycloudflare.com/ws. Open it in Chrome or Edge.
  2. Look at the bottom-left LED. CONNECTED = good. If DISCONNECTED, the driver's tunnel isn't up — radio them.
  3. Press F11 for full-screen.
  4. If the URL changes mid-race (the driver restarted their tunnel), press ⚙ CONNECTION and paste the new one.

SCREEN LAYOUT

  • Header — track · car · driver · class · weather · session clock.
  • Left column (VITALS) — gear, RPM, throttle/brake/clutch, steering, speed.
  • Middle column (TIRES + CAR STATUS) — 4 wheels with temps + fuel/water/oil/stint.
  • Right column (TIMING TOWER) — position, lap times, sectors with deltas, gaps, flag strip.
  • Far-right column (CLASSES + MAP) — full multiclass standings and live track map with every car.

WHAT TO RADIO (TYPICAL CALLS)

Fuel running low

  • FUEL card turned orange → "Pit window opens in 3 laps, fuel."
  • FUEL card red + blinking, or FUEL! flag lit → "Box this lap, fuel critical."

Tires degrading

  • All 4 yellow / WEAR below 40% → "Manage tires, save the rears."
  • Any red → "Tires gone, box for new set."

Engine hot

  • WATER > 105 °C → "Engine hot, lift on straights."
  • OIL > 130 °C → "Box immediately, oil critical."

Faster class catching up (blue flag)

  • On the MAP a red dot (Hypercar) approaches your yellow GT3 from behind.
  • → "Hypercar 5 seconds behind, blue flag coming."
  • When clear → "Track ahead clear, push."

Competitor pits

  • In CLASSES table the row near yours shows PIT.
  • → "P2 in pit, you gain position next sector."

Pace dropping

  • LAP TIMES — BEST stable but LAST drops 0.5-1.5 s lap after lap.
  • → "Pace dropping, tires going, plan stop in N laps."

Frontend hosted away from the Python server (e.g. Cloudflare Pages, GitHub Pages, Base44) needs to know where the telemetry backend is. Get the public URL from ngrok http 8000 on the racing PC and append /ws.

Example: wss://lmu-strat.ngrok-free.app/ws

You can also pass it once via the query string:
?ws=wss://...ngrok-free.app/ws — it'll be remembered after that.
